# Archivos

Los archivos PDF son una excelente manera de proporcionar información detallada a tu chatbot Talkee. Aquí te explicamos cómo usar esta función:

## ¿Cómo subir un archivo PDF?

1. Haz clic en el área de "Drop your files here" o selecciona "Subir PDF".
2. Elige el archivo PDF que deseas subir desde tu computadora.
3. Una vez seleccionado, haz clic en "Subir PDF" para añadirlo a tu chatbot.

## Consejos para usar archivos PDF:

- Asegúrate de que los PDF contengan información relevante y actualizada.
- Usa PDF con texto seleccionable para mejores resultados.
- Puedes subir varios archivos PDF para cubrir diferentes temas.

## Gestión de archivos PDF:

- Verás una lista de los PDF subidos.
- Puedes eliminar un PDF haciendo clic en el icono de papelera junto a su nombre.

Recuerda: Después de subir nuevos PDF o eliminar existentes, deberás reentrenar tu chatbot para que incorpore estos cambios.

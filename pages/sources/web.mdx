# Web

La función de Contenido Web te permite importar información directamente de páginas web a tu chatbot Talkee. Es una forma rápida de añadir grandes cantidades de información actualizada.

## Cómo añadir contenido web:

1. En el campo "URL", introduce la dirección web de la página que quieres importar.
2. Haz clic en "Buscar información" para que Talkee extraiga el contenido de la página.
3. Revisa y edita el texto extraído en el área de texto grande si es necesario.
4. Cuando estés satisfecho con el contenido, haz clic en "Añadir información".

## Consejos para usar el contenido web:

- Elige páginas web con información relevante y confiable.
- Edita el contenido extraído para eliminar información innecesaria o irrelevante.
- Utiliza múltiples páginas web para cubrir diferentes aspectos de tu negocio o servicio.

## Gestión del contenido web:

- Puedes ver una lista de las URLs que has añadido previamente.
- Para cada URL, puedes ver el contenido extraído y editarlo si es necesario.
- Tienes la opción de eliminar contenido web que ya no sea relevante.

Recuerda: El contenido web puede cambiar con el tiempo, así que es buena idea revisar y actualizar periódicamente la información importada de páginas web.

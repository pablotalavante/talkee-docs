# Integración

¡Felicidades por haber configurado tu talkee! Ahora es el momento de compartirlo con el mundo. Esta sección te muestra cómo puedes integrar tu chatbot en tu sitio web o compartirlo como un enlace independiente.

## Opciones de Integración

Cuando haces clic en el botón "Integración", verás dos opciones principales para compartir tu chatbot:

### 1. Integración en tu Sitio Web

Para añadir tu chatbot directamente a tu sitio web, usa el siguiente código:

```html
<script src="https://api.talkee.co/api/widget/" type="text/javascript" data-chatbotid="chatbot-id" defer></script>
```

**Cómo usar este código:**
1. Copia el código proporcionado.
2. Pega este código justo antes de la etiqueta de cierre `</body>` en el HTML de tu sitio web.
3. El chatbot aparecerá como un widget en la esquina inferior derecha de tu sitio.

**Nota:** El `data-chatbotid` en el código es único para tu chatbot. No lo cambies, ya que este ID es el que conecta el widget con tu chatbot específico.

### 2. Enlace Independiente

Si prefieres compartir tu chatbot como una página web independiente, puedes usar este enlace:

```
https://talkee.co/iframe/chatbot-id
```

**Cómo usar este enlace:**
- Comparte este enlace directamente con tus usuarios.
- Puedes incluirlo en correos electrónicos, mensajes o cualquier otra comunicación.
- Al hacer clic en el enlace, se abrirá una página web dedicada con tu chatbot.

## Consejos Adicionales

- **Prueba la integración**: Después de añadir el código a tu sitio, asegúrate de probarlo en diferentes páginas y dispositivos.
- **Personalización**: Recuerda que puedes ajustar la apariencia de tu chatbot en la sección de "Interfaz" para que combine perfectamente con tu sitio web.
- **Actualizar**: Si haces cambios en tu chatbot, estos se reflejarán automáticamente en todas las integraciones existentes.
